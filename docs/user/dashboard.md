## Key Sections

### 1. Welcome Banner (First Visit)

Shows:
- Welcome message with your name
- Quick start options
- Tips for getting started
- Can be dismissed

### 2. My Projects

**Shows**: Projects you're involved with

**Includes**:
- Project name and description
- Current status (Planning, In Progress, Testing, Deployed)
- Progress percentage
- Key dates (start, target completion)
- Team members involved
- Quick actions (View, Edit, Update Status)

**Status Colors**:
- ğŸ”µ Blue: Planning phase
- ğŸŸ¡ Yellow: In Progress
- ğŸŸ  Orange: Testing phase
- ğŸŸ¢ Green: Deployed/Complete

**Click a project** to view full details and manage it.

### 3. Recent Activity

**Shows**: Recent actions in your organization

**Types of Activities**:
- New use cases submitted
- Projects status updated
- Milestones achieved
- Team members added
- Comments and discussions

**Timeline format** with most recent at top.

### 4. Assigned Use Cases

**Shows**: Use cases you're assigned to

**Information per use case**:
- Title and description
- Your role (Sponsor, Developer, Owner, etc.)
- Current phase
- Due date
- Priority level (Low, Medium, High, Critical)

**Actions**:
- View full details
- Update progress
- Add comments
- Manage team

### 5. Key Metrics

**Dashboard metrics** showing:

| Metric | Description |
|--------|-------------|
| Total Initiatives | All AI use cases in organization |
| Active Projects | Currently in progress |
| Completed | Successfully deployed |
| Success Rate | % of completed vs total |
| Avg Time to Deploy | Average duration |
| ROI Impact | Calculated business value |

Click any metric for detailed analytics.

### 6. Action Items

**Your pending actions**:
- Approvals needed
- Information to provide
- Updates due
- Team responses needed
- Comments requiring response

Click item to address immediately.

## Customization

### Rearrange Dashboard Cards

1. **Hover** over a card
2. Click the **three dots** menu (â‹®)
3. Select **"Move"** or **"Resize"**
4. Changes auto-save

### Show/Hide Sections

1. Click **"Customize Dashboard"** (top right)
2. Toggle sections on/off
3. Arrange in preferred order
4. Click **"Save Changes"**

### Filter Projects

In the **My Projects** section:
- Use **Filter** dropdown
- Filter by: Status, Team, Phase, Priority
- Search by project name
- Combine multiple filters

### Sort Options

- By Date (newest/oldest)
- By Status
- By Priority
- By Progress
- Alphabetical

## Common Actions

### View Project Details

1. Click on a project card
2. Opens full project page
3. View all information
4. Manage settings

### Update Project Status

1. Click **"Update Status"** on project card
2. Select new status from dropdown
3. Add optional comment
4. Click **"Update"**

### Add Comment to Activity

1. Click on activity item
2. Scroll to "Comments" section
3. Type your comment
4. Click **"Post Comment"**

### Quick Search

1. Press **Ctrl+K** / **Cmd+K**
2. Type search term
3. Select from results
4. Navigate to item

### Access Your Profile

1. Click your name/avatar (top right)
2. Options:
   - View Profile
   - Edit Settings
   - Preferences
   - Sign Out
